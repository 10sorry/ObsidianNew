12 КОМАНД GIT

git init
Инициализирует новый Git-репозиторий в текущей директории.
$ mkdir railsapp
$ cd railsapp
$ git init
git add
Добавляет изменения в текущей директории и вложенных папках в индекс (staging area).
$ echo "# RAILS APP" > README.md
$ echo "Demo" >> README.md
$ git add README.md
git commit
Фиксирует изменения из индекса с сообщением коммита.
$ git add app.css
$ git commit -m "Initial commit"
git push
Отправляет локальные изменения в удалённый репозиторий.
$ git add .
$ git commit -m "Update files"
$ git push origin main
git pull
Получает изменения из удалённого репозитория и сливает их в текущую ветку.
$ git status
$ git pull origin main
$ git log --oneline
git remote
Добавляет удалённый репозиторий, отображает список удалённых, переименовывает их.
$ git remote add origin <url>
$ git remote -v
$ git remote rename origin upstream
git branch
Показывает список веток, создаёт новую ветку и проверяет её наличие.
$ git branch
$ git branch feature-login
$ git branch
git fetch
Получает последние данные из удалённого репозитория, но не вносит их в текущую рабочую директорию.
$ git fetch origin
git checkout
Переключается на указанную ветку.
$ git branch
$ git checkout feature-login
$ git branch
git merge
Объединяет указанную ветку с текущей (в данном случае — с main).
$ git checkout main
$ git merge feature-login
$ git log --oneline
git status
Показывает текущее состояние репозитория, включая незафиксированные изменения.
$ git status
$ echo "Data Added" >> hello.txt
$ git status
git reset
Откатывает текущую ветку к указанному коммиту.
$ git log --oneline
$ git reset --hard <commit-hash>
$ git status
