
# Отключение МКЦ

Дата: [[2025-09-10]]
Теги: #linux
Связи: [[linux]] [[bash]]

---

**Описание**:  
Выключение МКЦ(Мандатного Контроля Целостности) на файловой системе 

**Пример команды**:

```bash
# пример bash-команды
astra-mic-control disable
```

Перед выключением режима МКЦ рекомендуется снять атрибуты целостности с объектов файловой системы командой:

```bash
unset-fs-ilev
```
